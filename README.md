# GoaT Stack 🐐 (Go and Typescript)


## Requirements
- [Just](https://just.systems)
- [Golang](https://go.dev/)
- [pnpm](https://pnpm.io)
- [Node](https://nodejs.org/en)

## The Stack

### Backend

Golang server with Protocol Buffer API

### Frontend

- React
- Vite
- TanStack Query
- ConnectRPC
- TanStack Router
- Tailwind CSS

## How to get started 🚀

1. Init the project
```bash
just init
```

2. Launch the dev

```bash
just dev
```
